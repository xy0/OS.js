Tutorial: Create your own Theme

Create your own Style theme

Simply copy the default theme and modify how you want it: 

cp -r src/client/themes/styles/default src/client/themes/styles/mytheme
edit src/client/themes/styles/mytheme/metadata.json
edit src/client/themes/styles/mytheme/style.less
edit src/conf/112-themes.json # Or https://os.js.org/doc/manuals/man-configuration.html
grunt config themes:styles:mytheme

Then update the theme manifest and LESS manually

Create your own Icon theme

Simply copy the default theme and modify how you want it (remember to change the manifest file): 

cp -r src/client/themes/icons/default src/client/themes/icons/mytheme
edit src/client/themes/icons/mytheme/metadata.json
edit src/conf/112-themes.json # Or https://os.js.org/doc/manuals/man-configuration.html
grunt config themes:icons

Create your own Sound theme

Simply copy the default theme and modify how you want it (remember to change the manifest file): 

cp -r src/client/themes/sounds/default src/client/themes/sound/mytheme
edit src/client/themes/sound/mytheme/metadata.json
edit src/conf/112-themes.json # Or https://os.js.org/doc/manuals/man-configuration.html
grunt config themes:resources

Create your own Font theme

Just add your fonts in a sub-folder like Karla. Then update the included stylesheet with correct name(s). 

edit src/conf/112-themes.json # Or https://os.js.org/doc/manuals/man-configuration.html
grunt config themes:fonts

Update your changes

Just run the same grunt commands as when you created the themes :)
