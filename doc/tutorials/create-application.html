Tutorial: Create your own application

Create from template

Simply run:

grunt create-package:default/MyName

The "default" is the name of the repository folder you want to place your package in.

You will end up with "main.js", "main.css" (for client), "api.js" (for server) and "metadata.json" (see below)

This will create some source files for you with usable code. You'll have a basic running application with a Window.

Update metadata

The metadata file holds general information about your application and stuff used in the building process.

You don't have to do this, but this shows what values you can put in

For more information on this see here.

Install

Now run this command to add it to global manifest (make it visible in OS.js)

grunt manifest config packages:default/MyName

If you are using "dist-dev", you don't have to run the "packages" grunt command.

Tips

If you want to reaload your application when it has been updated simply run "OSjs.API.relaunch('ApplicationMyName')" in the developer console of your browser.
