Tutorial: Create IFrame Application

Very useful if you want to convert something you already made or don't want to use the OS.js API.

Create Package

grunt create-package:default/MyPackageName:iframe

The "default" is the name of the repository folder you want to place your package in.

You will end up with "main.js" (for client) and "metadata.json" (see below)

Update metadata

See Application Tutorial page for more information on this.

Code your Application

Simply create the directory "data" in your application directory (src/packages/MyPackageName) with the file "index.html".

Then go ahead and write your app in the "index.html" file.

Bi-directional communication

You can communicate between OS.js and your IFrame application using the browser APIs and some helper methods.

  • API (please not that this is updated at times. It will be available via repository soon)
  • Application Example
  • Iframe markup Example

Including resources

If you want to include resources into your application, make sure to use relative paths. 

  <script src="mylib.js"></script>

  <link rel="stylesheet" type="text/css" src="mylib.css" />

Tips

If you want to reaload your application when it has been updated simply run "OSjs.API.relaunch('ApplicationMyName')" in the developer console of your browser.
